# be-bot-api

## Install libs

```bash
 pip install -r requirements.txt
```

## Start project

```bash
 uvicorn be_bot_api.api.main:app --host 0.0.0.0 --port 8000
```

## Virtual Enviroment

### Create new enviroment
```bash
 python -m venv venv
```

### Activate on windows
```bash
 venv\Scripts\activate.bat
```
### Activate on linux
```bash
 source venv/bin/activate
```
